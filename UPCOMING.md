# Upcoming Game Engine Features

The roadmap for Match Logic, Statistics, and Algorithms.

- [ ] **Multi-Game Support**: Architecture to support Ping Pong, Chess, and Foosball alongside Pool.
- [ ] **Tournaments**: automated bracket generation (Single Elimination, Round Robin).
- [ ] **Advanced Stats**:
    - "Nemesis" tracking (who you lose to the most).
    - Longest Win/Loss streaks.
    - Win rates by time of day.
- [ ] **Season Management**: Ability to archive stats and start a fresh "Season 2".
- [ ] **Dispute Resolution**: Logic for handling contested match results.
- [ ] **Achievements**: badges for milestones (e.g., "Rookie", "Slayer", "Unstoppable").
